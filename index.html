📠 runml (C Mini-Language Transpiler)

A command-line utility written in C11 that compiles and executes programs written in a custom-designed mini-language (.ml). Runml transpiles high-level .ml source code into C11, compiles the result, and executes the output, all within a single utility.

  • ⚙️ Written in standard C11 with POSIX system calls (no third-party dependencies)
  • 🔁 End-to-end execution: transpilation ➝ compilation ➝ execution ➝ cleanup
  • 💻 Demonstrates low-level systems knowledge: process management, I/O, and compilation automation

💊 Modeling Doctor Complaints with Zero-Inflated Negative Binomial Regression

A statistical modeling project analyzing doctor complaints in emergency services using advanced count regression methods. In this project, I used Zero-Inflated Negative Binomial (ZINB) modeling to handle overdispersed and zero-inflated count data. The model uncovered how workload, income, and gender/residency interactions affect complaint frequency.

  • 📊 Modeled overdispersed count data using Poisson, Negative Binomial, ZIP, and ZINB approaches
  • 🧠 Applied Zero-Inflated Negative Binomial with interaction terms as final model
  • 🧪 Performed residual diagnostics, AIC comparison, and visual model checks
More

🚦 Data Warehouse Design and Implementation for Road Safety Analytics

A real-world data warehousing and analytics project built to support government decision-making on road safety in Australia. The project involved designing and implementing a dimensional model, building a PostgreSQL-based data warehouse, and generating interactive insights using Power BI and association rule mining in Python.

  • 🧱 Designed a star schema with 8+ dimensions and a central fact table for fatal crash events
  • 📊 Built dynamic dashboards in Power BI to visualize regional crash patterns and trends
  • 🧼 Implemented robust ETL with Pandas to clean, transform, and load real-world datasets
  • 🔍 Applied association rule mining to uncover high-risk patterns among road user groups
  • 🧠 Proposed data-driven safety recommendations for policymakers based on mined insights

🧩 Graph Database: Road Crash Graph Insights (Neo4j)

A property graph database project focused on fatal road crashes in Australia. Using Neo4j and Cypher query, my report captures complex interactions among crashes, people, vehicles, and locations to reveal actionable insights using graph traversal and pattern matching.

  • 🧠 Designed a normalized property graph using Arrows App to model crash networks
  • 🛠️ Performed ETL from tabular data to CSV node/edge files for Neo4j ingestion
  • 🔍 Executed Cypher queries to extract patterns by time, vehicle type, gender, and geography
  • 📈 Explored advanced queries like multi-hop paths and peak-hour regional analysis
  • 🌐 Investigated real-world applications of graph data science in road safety

Ria Raya!

Ria Raya is argueably hands-down the biggest event the club has ever hosted since the creation of the club, and one we’re incredibly proud when looking back to. It is our way of celebrating the richness of Malaysian culture, with a special focus on the traditions surrounding Hari Raya. The evening comes alive with food trucks serving up all-time Malaysian favourites, from satay to nasi lemak. We also feature traditional games like congkak and batu seremban, as well as cultural performances such as silat demonstrations and Malay dance. It’s a vibrant celebration that brings people together no matter the nationalities, to share in the food, fun, and festive spirit.
